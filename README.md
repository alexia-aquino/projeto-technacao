<p>A interace consiste em uma tela de login fullstack de uma instituição fictícia de ensino digital 
desenvolvida para uma avaliação da disciplina Desenvolvimento Backend I, no curso de Análise e Desenvolvimento de Sistemas da Universidade Veiga de Almeida.</p>

<h2>Tecnologias:</h2>
<ul>
  <li> <b>Backend:</b> Node.js + Express + TypeScript</li>
  <li> <b>Frontend:</b> Vue.js 3 + TypeScript + Vite</li>
  <li> <b>Design:</b> Adobe Photoshop + Adobe Illustrator + Aplicações de UX/UI</li>
  <li> <b>Hospedagem:</b> Vercel (Frontend) + Render (Backend)</li>
</ul>

<br>

<h2>Funcionalidades:</h2>
<ul>
  <li> Listagem de cursos via API; </li>
  <li> Formulário de matrícula com validação; </li>
  <li> Comunicação entre Frontend e Backend; </li>
  <li> Responsividade aplicada para telas menores. </li>
</ul>

<br>

<h2>Aprendizados:</h2>
<ul>
  <li> Consumo de APIs REST; </li>
  <li> Validação de dados no Backend; </li>
  <li> Comunicação entre Frontend e Backend; </li>
  <li> Deploy de aplicações Fullstack. </li>
</ul>
